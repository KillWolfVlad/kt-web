# kt-web

kt-web это неофициальная веб-версия KinoTrend

## Цель проекта

Загрузить данные из заданного пользователем источника и показать список фильмов

### Особенности по сравнению с официальной веб-версии KinoTrend

- Показывать магнет-ссылки на фильмы (это абсолютно легально, всю ответственность несет пользователь, т.к. он указывает источник с данными, а приложение просто отображает их)
- Поддержка WebHooks, чтобы пользователь мог при нажатии одной кнопки в приложении отправить запрос на свой сервер и начать скачивать фильм

## Технический стек

- SPA приложение (с поддержкой PWA)
- React + TypeScript (последних версий)
- Поддерживаемые браузеры: все современные браузеры (Chrome + Firefox + Safari)
- Поддерживаемые устройства: телефоны, планшеты, компьютеры (в том числе с Retina дисплеями)
- Для сборки проекта используется `vite`
- Пакетный менеджер: `npm`
- Для форматирования используется `prettier`
