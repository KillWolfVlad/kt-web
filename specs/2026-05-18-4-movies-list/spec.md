# 2026-05-18-4-movies-list

- **ВАЖНО**: `Привет мир!` больше не нужно отображать, вместо него должен отображаться список фильмов
- Если источник с данными для загрузки списка фильмов (`data.json`) не задан, то ничего не отображай (т.к. поверх будет отображаться диалоговое окно с настройками)
- Когда Источник с данными для загрузки списка фильмов (`data.json`) меняется, то необходимо с него `GET` запросом загрузить список фильмов
- Пока идет загрузка необходимо отобразить spinner в виде крутящегося по часовой стрелке эмодзи `🎬` по центру экрана
  - Размер эмодзи - `H1`
- В случае ошибки необходимо отобразить текст ошибки в красной рамке по центру экрана
- Схему данных смотри в `rules/data-schema.md`
- Отобрази фильмы в виде сетки с небольшим отступом сбоку и сверху
  - Используй умный подбор количества элементов в строке сетки в зависимости от ширины экрана

## Состав элемента сетки

- Весь элемент состоит из `Movie.posterURL` (все постеры имеют ширину 240px и разную высоту, высота должна быть выровняна у всех элементов и быть одинаковой)
- Внизу элемента поверх изображения на черном (50% прозрачности фоне) должны отображаться:
  - `[flag] [Movie.nameRU]` если `flag` можно вычислить и просто `[Movie.nameRU]` если `flag` нельзя вычислить
  - `⭐ [Movie.rating]`

### Алгоритм вычисления `flag`

Входные данные: `Movie.country`
