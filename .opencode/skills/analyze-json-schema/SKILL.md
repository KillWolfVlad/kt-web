---

## What I do

- Запроси у пользователя ссылку на JSON файл, который нужно проанализировать
- Скачай файл и сохрани его в `.opencode/tmp`
- Создай файл `.opencode/tmp/.gitignore` с содержимым `*` c целью исключить директорию из коммитов
- Проанализируй схему скаченного файла и сохрани ее в `.opencode/rules` (запроси у пользователя название для правила, например `user-data-schema`)
  - Укажи типы данных для всех полей и их обязательность
