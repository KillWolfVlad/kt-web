## What I do

- Скачай файл <https://example.com/data.json> и сохрани его в `.opencode/tmp/data.json`
- Создай файл `.opencode/tmp/.gitignore` с содержимым `*` c целью исключить директорию из коммитов
- Проанализируй схему файла `.opencode/tmp/data.json` и сохрани ее в `.opencode/rules/data-schema.md`
  - Укажи типы данных для всех полей и их обязательность
