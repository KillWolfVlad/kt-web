# Дизайн-система

Система построена на двух цветах из favicon: `#1a1a2e` (фоновый тёмно-синий) и `#e94560` (акцентный красный).

## Правила использования

При разработке компонентов обязательно использовать CSS-переменные из этой дизайн-системы. Запрещено хардкодить цвета, скругления, тени, отступы и другие токены напрямую.

## Цветовая палитра

### Тёмная тема (основная / по умолчанию)

| Токен | Значение | Описание |
|-------|----------|----------|
| `--color-bg` | `#1a1a2e` | Основной фон |
| `--color-bg-elevated` | `#16213e` | Фон карточек, модалок |
| `--color-bg-hover` | `#1a2744` | Фон при наведении |
| `--color-primary` | `#e94560` | Акцентный цвет (кнопки, ссылки, активные элементы) |
| `--color-primary-hover` | `#d63850` | Акцентный цвет при наведении |
| `--color-text` | `#ffffff` | Основной текст |
| `--color-text-secondary` | `#a0a0b8` | Второстепенный текст (подписи, метаданные) |
| `--color-border` | `#2a2a4a` | Границы элементов |
| `--color-success` | `#2ecc71` | Успех (сиды, доступно) |
| `--color-warning` | `#f39c12` | Предупреждение |
| `--color-error` | `#e74c3c` | Ошибка |

## Типографика

| Свойство | Значение |
|----------|----------|
| Font-family | системная (`-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif`) |
| Моноширинный шрифт | `'SF Mono', 'Fira Code', 'Cascadia Code', monospace` |

## Скругления

| Токен | Значение | Применение |
|-------|----------|------------|
| `--radius-sm` | `6px` | Мелкие элементы (бейджи, чипсы) |
| `--radius-md` | `12px` | Карточки, инпуты, кнопки |
| `--radius-lg` | `16px` | Модальные окна, крупные блоки |

## Тени

| Токен | Значение | Применение |
|-------|----------|------------|
| `--shadow-sm` | `0 1px 3px rgba(0,0,0,0.3)` | Карточки, элементы поверх фона |
| `--shadow-md` | `0 4px 12px rgba(0,0,0,0.4)` | Модальные окна, дропдауны |
| `--shadow-lg` | `0 8px 24px rgba(0,0,0,0.5)` | Крупные модалки, оверлеи |

## Анимации

| Токен | Значение | Применение |
|-------|----------|------------|
| `--transition-fast` | `150ms ease` | Ховеры, состояния |
| `--transition-normal` | `250ms ease` | Появление/исчезновение элементов |
| `--transition-slow` | `400ms ease` | Страницы, крупные переходы |
