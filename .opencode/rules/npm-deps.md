# Правила работы с npm зависимостями

- При установке npm пакетов всегда сначала читай `package.json`
- Устанавливай `dependencies` и `devDependencies` отдельными командами:
  - Сначала `npm install <pkg>` (обычные зависимости)
  - Затем `npm install -D <pkg>` (dev-зависимости)
- После каждого `npm install` проверяй `package.json` — зависимости должны сохраниться в правильной секции
- Никогда не используй `npm install -D` и `npm install` (без `-D`) в одной команде или параллельно — это может привести к потере записей в `package.json`
- Всегда фиксируй (пинь) версии зависимостей: используй `npm install --save-exact` или `npm install -E`, чтобы в `package.json` версии сохранялись без `^` и `~`
